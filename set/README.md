Реализация коллекции типа множества (set)

## Ограничения
Доступное подмножество языка: функции, ассоциативный массив (map)

## Интерфейс
```go
set(slice) // создание множества
get(set, elem) // получить элемент множества
has(set, elem) // проверить наличие элемента в множестве
append(set, elem) // добавить элемент в множество
delete(set, elem) // удаление элемента из множества с помощью built-in функции (delete)
clear(set) // удаление всех элементов в множестве с помощью built-in функции (clear)
len(set) // количество элементов в множестве с пощью built-in функции (len)

union(S,T) // объединение множеств (S ∪ T)
intersection(S,T) // пересчение (S ∩ T)
difference(S,T) // разница множеств (S - T) = (S \ T)
symmetric_difference(S,T) // симметрическая разность множеств (S △ T) = (S ∖ T) ∪ (T ∖ S)

```

## Демо

```go
s := set([]int{1,2,2,2,3,4,5,1,0,0,1,4}) // s = [0,1,2,3,4,5]
get(s, 5) // 5, true
has(s, 0) // true
append(s, 5) // s = [0,1,2,3,4,5]
append(s, 6) // s = [0,1,2,3,4,5,6]
delete(s, 6) // s = [0,1,2,3,4,5]
clear(s) // s = []

s = set([]int{1, 2, 3, 4, 5})
t := set([]int{3, 4, 5, 6, 7})
union(s, t) // 1,2,3,4,5,6,7
intersection(s, t) // 3,4,5
difference(s, t) // 1,2
symmetric_difference(s, t) // 1,2,6,7
```
